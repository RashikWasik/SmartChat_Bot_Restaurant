# Restaurant SmartChat Bot

<img width="1027" height="575" alt="image" src="https://github.com/user-attachments/assets/325c69a2-2fb6-43b0-84b6-2837f09aca58" />

Features:
1. AI Agent that understands and replies to customer messages.
2. Connected to Google Sheets with 3 sheets: Available Items – list of food items currently available, Common Q&A – answers for frequently asked questions (refund policy, timings, etc.), Customer Requests – blank sheet for storing new customer queries.
3. Reads food availability from the Available Items sheet and responds instantly.
4. Answers customer questions using information from the Common Q&A sheet.
5. If the customer asks something not found in the Q&A sheet or requests an unavailable item: The agent asks for the customer’s phone number. Saves the question + phone number into the Customer Requests sheet.
